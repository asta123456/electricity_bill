# Electricity Bill Prediction using Machine Learning

## Project Overview

This project predicts electricity bills using Machine Learning Regression algorithms.

The application is built using:

- Python
- Scikit-learn
- Streamlit

The model predicts electricity bill amounts based on user input features.

---

# Features

- Data preprocessing
- Missing value handling
- Feature scaling
- Multiple regression algorithms
- Model evaluation
- Streamlit web application
- Real-time prediction

---

# Algorithms Used

- Linear Regression
- Decision Tree Regressor
- Random Forest Regressor
- Support Vector Regressor
- KNN Regressor
